This is a test of multimedia and how the server can stream it to the client. This is done by figuring out the correct MIME type, and then sending the data to the browser as raw bytes.

There are various examples here, all of which don't infringe on copyright laws.

Extension MIME Content Bitrate Sample
.ogg audio/ogg Kick Sample 112 kbps
.mp3 audio/mpeg iDestyKK - Moonlight (Sample) 320 kbps
.wav audio/wav Card Draw Lossless

Supported MIME types:

  • audio/wave
  • audio/mpeg
  • audio/flac
  • audio/webm
  • audio/ogg

You may also view the audio files directly on the server and download them. The server supports it.
